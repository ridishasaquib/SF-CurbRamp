About SF Curb Ramp

SF CurbRamp is an interactive map designed to visualize the accessibility of curb ramps in San Francisco. Use this application to locate curb ramps, report missing ramps, and explore data on their condition.

The map provides valuable data for planners, policymakers, and advocates to identify areas that need immediate attention. It encourages community-driven initiatives to improve infrastructure and create a more inclusive environment for everyone.

This project promotes inclusivity by identifying accessibility gaps, helping ensure public spaces are safer and more navigable for individuals with disabilities, fostering equal access for everyone.
